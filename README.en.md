# 人脸识别身份认证系统

#### Description
在“信息系统与安全对抗技术”上作为结课代码提交的文件，现开源于此，供大家参考。本项目采用Python语言开发，在跨年龄识别功能中引用了Paddle模块，对除跨年龄识别模块的部分进行了Windows安装包打包。项目中仍有诸多缺陷，请大家有何需求和建议请在issue中讨论，时间和能力允许范围内我会予以解答和完善。
